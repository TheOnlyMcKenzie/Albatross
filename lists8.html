LISTS

Filling a 2D List

If we want to fill a 2D list, we need a nested loop. Additionally, inside each loop we would need to create a new list.

This is the end of the Lists course. If you are keen to learn more, check out our other courses!

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

x_list = []
count = 0

for i in range(2):
	y_list = []
	x_list.append(y_list)
	for j in range(2):
		y_list.append(count)
		count += 1
