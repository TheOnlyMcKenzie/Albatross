DICTIONARIES

Adding to a Dictionary

To add a single item to a dictionary, write the key in [key] and assign the value, like so: country_foods["Mexico"] = "Burrito".

If we want to add multiple items to our dictionary, we can use the .update() function, writing the contents we want to add to the dictionary just as if we were adding a new one. 

country_foods.update({"France":"Croissant", "Thailand":"Phad Thai"})

Changing Values in a Dictionary

We can modify a single value by writing something like this: country_foods["Mexico"] = "Taco". We have changed Mexico's value from "Burrito" to "Taco".

We can modify multiple values the same way we use update(), this will update any new items to its new value and if it does not exist, add it to the dictionary. 

country_foods.update({"France":"Bouillabaisse", "Spain":"Paella"})

Removing Values from a Dictionary

Finally, just as with lists, we can use the del function to remove a value from a dictionary, but instead of passing the index, we pass the key.

del country_foods["France"]

Be careful that you don't forget to include the key - otherwise you'11 delete the entire dictionary!
