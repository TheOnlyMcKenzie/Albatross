LISTS

Sorting a List

Sometimes we need to sort things in our list. We do this using the .sort() function. They have to be the same datatype, though - you can't sort a heterogeneous list.

Capitals come before lowercase.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

foods = ["Pizza", "Ice Cream", "Apple", "Cherry", "Pie", "Steak"]

foods.sort()

print(foods)

Shuffling a List

For some reason we may want to shuffle our list. We will need to import from the random library to do so. From there we use random.shuffle().

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

import random

foods = ["Pizza", "Ice Cream", "Apple", "Cherry", "Pie", "Steak"]

random.shuffle(foods)

print(foods)

Split

We can split a string on a given token to convert it into a list using .split()

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:


string = "Here is a sentence"

string_list = string.split(" ")

print(string_list)
