FUNCTIONS

Getting Data Out Of a Function

Just as a function can accept values (arguments), a module can also be used to calculate and return those values to where they were called. This is done by using the keyword return followed by one or more values.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

def request_info():
	"""Ask user for exercise information
	   Then returns three values
	"""

	exercise = input("What was the exercise?: ")
	weight = float(input("How heavy was the weight in kgs?: "))
	reps = int(input("How many reps did you do?: "))

	return exercise, weight, reps

user_exercise, user_weight, user_reps = request_info()
print(user_exercise, user_weight, user_reps)
