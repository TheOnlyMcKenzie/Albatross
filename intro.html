PYTHON BASICS - INTRODUCTION

Python is a general-purpose programming language created by Dutch computer scientist (and Monty Python fan) Guido van Rossum in the early 1990's. It has gone on to become one of the more widely-used programming languages, and its relatively simple rules and syntax makes it a good language to use for learning how to code in general.

If you are new to Python or programming in general, then here is the right place. In this course we will cover the fundamentals of programming in Python - the print statement, variables, loops (for and while) and conditionals (if/else statements).

Each lesson will have some example code. Provided you have IDLE (or any other IDE) installed, you can copy this code to see what it does and play around with it.
