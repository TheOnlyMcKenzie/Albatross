LISTS

Slices

With a list, often we will not need to use the whole thing. We might want to take only a subsection of it. A subsection of a list is called a slice.

Taking a slice out of a list is like how we use the range function. We need to define the start index and the end index. Also like a range, the start index is inclusive and the end index is exclusive.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

foods = ["Pizza", "Ice Cream", "Apple", "Cherry", "Pie", "Steak"]

print(foods[2:5])
