Getting Data Into a Function

The real power of computing starts to be unleashed when we're able to apply a function repeatedly to different pieces of data. The data might be stored somewhere that the function can access it such as a function accessing global variables. Usually, however, it is better design to send the input data directly into the function in the form of one or more inputs specified in its parentheses. These inputs are usually called parameters or arguments.

This works in the same way as, for example, the sine function on a calculator “sin(x)”. The sine function must be sent an input to work with. In fact, we have been sending inputs to a function right from our very first program. Specifically we have been sending the data that we want to print out to the print() function e.g. print("Hello").

The function definition must create in its parentheses ( ) one or more local variables for each input it is being set up to receive. These input parameters operate as local variables, only accessible within the function body. 

 
def function_one(s):
	"""set up for one input
	   names it s
	"""
	print(s)

In the main routine, this function could be called as follows: 

 
function_one("Hello World")
