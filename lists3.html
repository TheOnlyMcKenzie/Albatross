Looping Through a List

Since lists are defined by order, one thing we can do with lists is loop through them to put each value in the list through a certain block of code/function, using a for loop.

We can do this by using the function len(). "len" means the length of a list, so if we run the loop for i in range(len(list)):, whatever code is in the loop will run the same amount of times as the amout of values in the list.

There is a short way of doing this, however it can't be applied in the same amount of circumstances that the long method can. To loop through a list we can just write for food in foods: (which is a shorter way of writing for i in range(len(foods)).

Example Code

The long method:

foods = ["Pizza", "Ice Cream", "Apple"]

for i in range(len(foods)):
	print(i)

The short method:

foods = ["Pizza", "Ice Cream", "Apple"]

for food in foods:
	print(food)
