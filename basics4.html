PYTHON BASICS

Collecting Data from User Input

Aside from hardcoding data into a variable, we can store user-entered data (or any externally-sourced data) in a variable. We do this using the input() statement.

The general format for an input statement is "(variable) = input(text)". The input prompt printed out to the user can either be a hard-coded string or a variable.

Outputting Data Using .format()

Additionally, we can output data in a neater fashion using .format(). This allows us to insert the value of one or more variables into a string to output.

This is done in the format "print('{}'.format(variable))". Remember to check that all brackets are closed - this is one of the most common errors in Python!

\

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

name = input("What is your name?")

print("Hello, {}".format(name))
