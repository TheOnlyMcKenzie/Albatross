FUNCTIONS - INTRODUCTION

Up until now, we have been doing everything inside the main module in our programs. This is fine when creating very basic programs and scripts, but in the real world, things are often much more complex.

Enter functions. In programming, a function refers to a block of code within a larger program that's built to perform a certain task e.g. calculate an area. The idea behind them is to make our programs more efficient - we don't want to repeat large sections of code every time we want to perform a certain function, because that wastes time and data. Instead, we define a function, write the code once, and then whenever that function needs to be performed we call upon it with only one line of code in the main routine.

In this course, we will be covering how to define and call a function in Python, as well as learning about local variables and how to pass data between different functions.
