DICTIONARIES

Looping through a Dictionary

There are two ways to do this - the keys() method along with the items() method (which returns both the key and the value).

Example Code

The keys() method:

for key in country_foods.keys():
	value = country_foods[key]
	print(key, "=", value)

The items() method:

for key, value in country_foods.items():
	print(key, "=", value)
