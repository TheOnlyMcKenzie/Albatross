PYTHON BASICS

Libraries

Like most programming languages, Python has many built-in libraries of code that can easily be imported into your programs.

The two libraries that most people learn about first are random and time. random lets you randomize things - for instance random.randint() generates a random number in a ceratin range (like in a for loop, the range is what goes inside the brackets).

time lets you bring elements of time into your program. For instance time.sleep() lets you delay a block of code from running by a certain number of seconds (the number of seconds is what goes in the brackets). Thinking in terms of usability, this is good to use to let the user know that something is happening in your program.

To import a library, simply type import followed by the name of the library.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

import random

num = random.randint(1,100)

for i in range(0,10):
	
	print(num)
