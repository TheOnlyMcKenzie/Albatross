LISTS

Creating a List

In Python, lists are denoted with square brackets [ ]. Everything inside those square brackets is part of the list. Items are separated by a comma.

There are two ways to create a list in Python. The first is to simply define the list like so: list = [item1, item 2..]

The other way is to use the list function. This is useful if we want to convert a string into a list. This is written as string_list = list(string). Note that the string must be contained within a variable in order for this to work.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

foods = ["Pizza", "Ice Cream", "Apple"]

print(foods)
