DICTIONARIES

Accessing Values in a Dictionary

We can use a dictionary to look up values by using our dictionary name and in square brackets we put a key (instead of using a number like in lists we put in the key's name) i.e. country_foods["Italy"].

Be careful when trying to access a key in a dictionary - if it doesn't exist you will get a key error. Take country_foods["germany"]. In this case, 'germany' does not exist because it is case sensitive (Germany).

Checking for a Key

To avoid looking foolish and getting a bug like that, you should first check by using an if-in condition to see if a key actually exists in the dictionary. 

country = "Germany"

if country in country_foods:
	print(country_foods[country])

Checking for a Value

What about a value in a dictionary? Well we can check for that too using .values().

food = "Hamburger"

if food in country_foods.values():
	print("That food exists in the dictionary")
