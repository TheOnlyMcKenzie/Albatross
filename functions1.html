FUNCTIONS

Defining a Function

A function has a definition - in Python this starts with the keyword def, followed by the function's name, a set of parentheses () (which may or may not be empty) and a colon : . This statement is sometimes called the header. The block of code which describes the behaviour of the function follows, indented by one tab (much like in a loop or if/else statement).

This indentation is not optional - Indentation is the way that Python and most other programming languages determine the structure of functions. 

def function_one():
	print("Albatross!")

Calling a Function

In order for a function to run it must be called, either in the main module or another function. To run the function above, we simply type function_one() into the main module.
