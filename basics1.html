PYTHON BASICS

The 'Print' Statement

In Python, the most basic and fundamental command is the print statement. All this does is print something onto the screen.

To print a line of text, use print(). Inside the brackets is where you put whatever you want to be printed. If it is a hard-coded string use quotes. You can wrap it in singular or double quotes, however it is generally best practice to use double quotes since single quotes also exist in English syntax with things like apostrophes, and if the text has been wrapped in single quotes the program will consider an apostrophe to be the end marker for the text, cutting it off short. Dialogue can be wrapped in single quotes.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

print("Hello, World!")
