PYTHON BASICS

Booleans

Boolean algebra is another aspect of mathematics which is centered around the idea that all values are either True or False. Booleans are instrumental in digital electronics and all modern programming languages, including Python, provide for them.

In terms of Python programming, True and False denote the condition of a boolean. If x is True, then y runs. If it is False, then z runs.

Booleans are particularly useful when running While Loops. The loop runs while x is True (or False depending on the context). Once a certain condition has been met, x becomes False (or True depending on the context) and the loop ends.

Following this logic, it is therefore possible to create an infinite loop using the statement while True:, and the only way to end this loop is to use break. This can be useful for some situations, however as mentioned before doing this is generally considered bad practice - it is best to avoid any situation where a loop may become infinte.

As you may have guessed by now, True and False are always written as proper nouns.

Boolean Operators

In addition to True and False, there also exist three boolean operators - they are and, or and not.

These can be used to narrow the conditions which one or more variables need to meet in order for a block of code to run e.g. if x > y and y < z:, or if x == a or y == b:.

Unlike True and False, boolean operators do not need to start with a capital letter.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

invalid = True

while invalid:
	age = int(input("Welcome to the Club! How old are you?"))

	if age >= 18:
		print("Come on in!")
		invalid = False

	elif age >= 0 and age < 18:
		print("Sorry, you're too young")
		invalid = False

	else:
		print("You can't have a negative age, silly!")
