DICTIONARIES - INTRODUCTION

In the previous course, we talked about lists. Lists are but one of many different types of arrays, a broader computer science concept relating to pieces of code that store data.

In this course we will cover another type of array in Python - Dictionaries. Unlike lists, one item in a dictionary is made up of two components - a key and a value. Like in lists, the keys and values in a dictionary can be of any datatype - strings, integers, variables, and yes, lists; the key's datatype also does not have to be the same as that of the value's. Also, unlike lists, order does not matter in a dictionary - if you print out the whole dictionary multiple times, the items in the dictionary will be in random order each time.

A common example of a program that uses a dictionary is a quiz. The program asks you a question - the key - and you enter the answer. If your answer corresponds to the question (key)'s value in the dictionary (the answer), then your answer is correct.
