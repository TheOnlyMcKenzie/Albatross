PYTHON BASICS

Conditionals (If/Else Statements)

The third essential component of Python we are going to cover are what computer scientists call conditionals. These are statements that describe the condition of a variable.

In most programming languages, including Python, conditionals are denoted with the statements if and else. The conditional in Python is written as "if (variable) =/>/< x:". Like in loops, the colon denotes an indented block of code that runs when the condition is met. If the condition is not met (else:), then another indented block of code runs.

There is also a midpoint between if and else called elif, which denotes an alternate block of code that runs when one condition is not met but another is (in this case the else runs if none of the conditions are met). Unlike if and else, you can have as many of these as you like between the opening if and closing else. In other programming languages, this statement is called else if or some other variation, but in Python the words are combined and it is written as elif.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

age = int(input("Welcome to the Club! How old are you?"))

if age >= 18:
	print("Come on in!")

else:
	print("Sorry, you're too young")
