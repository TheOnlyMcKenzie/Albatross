LISTS

2D Lists

A 2D List is a list of lists. We've mentioned before that lists are heterogeneous. This means that we can put any type of data into a list: strings, integers, floats,Booleans and yes, other lists. Putting a list inside a list makes a list 2-dimensional.

To create a 2D list you would write the notation like so: 

list_2d = [["a", "b"], [1,2,3]]

All the same things we just learnt about 1D lists also apply to 2D lists.

Accessing Items in a 2D List

If we want to access a value in our 2d list be careful! You don't write it with a square bracket inside a square bracket list how you create your 2d list. Instead you write it next to each other with the first index referring to the first index of the list and the second the second. 

print(list_2d[1][2])

Technically you could just keep going on with a 3D list etc. (a list of lists of lists... ListCeption much?)
