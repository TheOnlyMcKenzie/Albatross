Loops Part 2 - While Loops

Now that we have covered for loops, let's look at the other kind of loop in Python - While Loops.

While loops are generally used when you want to repeat something until a certain condition is met. There are ways to make while loops behave like for loops, but those are generally only used in specific circumstances.

When creating a while loop, we type "while (variable) =/>/< x:". Basically, while a variable is greater than, less than, equal to or not equal to a certain value (usually another variable or a constant), the block of code in the loop is repeated.

For Python loops in general, we write a colon after the loop command to denote indentation. One intented block of code is nested within the while loop by one tab.

A Note About the 'Break' Statement

To break any while loop, you can use the break statement (you can also use this in for loops). However it is generally considered bad practice to use break as it creates too much potential for bugs.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

num = 10

while num > 0:
	
	print(num)
	num -= 1
