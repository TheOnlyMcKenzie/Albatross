PYTHON BASICS

variables

If you know anything about algebra, you'll know that a variable is any value that changes. In programming, variables represent the backbone of functionality.

In Python, variables are denoted by a few conventions - variables are written in lower case, and if there are multiple words they are separated by underscores e.g. total_score.

When changing the value of a variable using any operators (plus, minus, multiply, divide, modulo etc.), add an equals sign to the operator to change it to your desired value e.g. total_score += 5 (this is really just a shorthand of writing total_score = (total_score + 5)).
