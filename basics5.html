Loops Part 1 - For Loops

In programming, loops are another essential component. A loop is where a block of code is repeated, either a certain number of times or until a condition is met. There are two main kinds of loops - for loops and while loops, and you can put any block of code in them, including other loops.

We will discuss for loops first. For loops are generally what to use if you want to repeat something a certain amount of times. To do this, we write "for (variable) in range(start, stop):".

When creating a general for loop, we use the variable i ("i" for integer). This represents the number of times we want our loop to run. Then we define our parameters using range(). The range is the difference between the stop and start point, so if we write for i in range(1,7):, the block of code indented within the loop will run six times. If only one argument is given in the range(), that will represent the stop point (by default the starting point is zero).

For Python loops in general, we write a colon after the loop command to denote indentation. One intented block of code is nested within the for loop by one tab.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

for i in range(1,7):
	
	print(i)
