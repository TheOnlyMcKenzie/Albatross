DICTIONARIES

Creating a Dictionary

In Python, dictionaries are denoted with curly brackets { }. Everything inside those curly brackets is part of the dictionary. Items (this time consisting of one key and one value) are separated by a comma. Keys and values are separated by a colon.

There are two ways to create a dictionary in Python. The first is to simply define the dictionary like so: dic = {key:value, key:value..}

The other way is to use the dict function. This is useful if we want to convert a list into a dictionary.

countries = ["Italy", "Japan", "USA", "Germany"]

foods = ["Pizza", "Sushi", "Hamburger", "Sauerkraut"]

country_foods = dict(zip(countries, foods))

print(country_foods)
