LISTS

Adding and Removing Items From a List

An important thing to remember is that like all arrays, lists start at zero. This means that the index (position) of the first item is 0, the second item is 1, the third item is 2 and so on.

Adding Items

If we know the index of the item we want to edit in the list we can change its value, just as we would do with a variable. We do it like this: foods[2] = "Hot Dog"

We can add a new value to the end of our list by using the append() function (foods.append("Cake")), and we can also choose where we can add an item into a list by using the insert() function. Unlike append() though, insert() takes 2 arguments, the index position and the value, in that order (i.e. foods.insert(2,"Banana")). Everything else in the list will shift as the list expands.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

foods = ["Pizza", "Ice Cream", "Apple"]

print(foods)

foods[2] = "Cherry"

print(foods)

foods.append("Pie")

print(foods)

foods.insert(1,"Steak")

print(foods)

Removing Items

If we know the index value of the item we are looking to remove we can use the del function (i.e. del foods[2]). The list will contract to account for the removed item.

Sometimes we may want to use the item from the list before we remove it. If we use the pop() function, this will remove the item at a specific index and return it (i.e. foods.pop(1)).

Popping an item from our list removes a value from the list but also returns it to us first. To do so, we need to know the index of the item we are looking to remove. We can set that returned value to a variable or just do nothing with it. The list will contract to account for the removed item.

Finally, we can remove an item from the list by its value, using remove() (i.e. foods.remove("Cake")). Warning, this only removes the first matching value.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

foods = ["Pizza", "Ice Cream", "Apple", "Cherry", "Pie", "Steak"]

print(foods)

del foods[5]

print(foods)

foods.pop(2)

print(foods)

foods.remove(1,"Pie")

print(foods)
