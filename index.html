WELCOME TO ALBATROSS

Welcome! Albatross is a website designed to teach the basics of the Python programming language. We currently offer simple tutorials on variables, loops, if/else statements, lists (including multidimensional lists), dictionaries and functions.

This website is designed to be most useful to people who have never programmed before. However, it should also be useful to anyone who has learned another programming language beforehand and wants to get started with Python.

To get started, simply choose one of the courses above!
