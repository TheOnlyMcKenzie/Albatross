LISTS

Looking in a List

When we loop through a list, we look at every item in the list. However in the real world things are often much more complex, and lists can get very long - imagine if the list contained over 200 foods; the program would be a little tedious for the user!

Checking to See If an Item is in a List

We can check to see if an item is in a list by doing the following: 

foods = ["Pizza", "Ice Cream", "Apple", "Cherry", "Pie", "Steak"]

food = input("What food would you like to check for?: ")
if food in foods:
	print("Yes it's there")
else:
	print("Sorry it's not there")

Handy, right? Additionally, since a string is essentially a list of characters (even though it does not behave like one), we can also apply the in function to a string.

Well now that we know whether it's in there or not, what if we also want to know where it is in the list?

Checking to see where an item is in a list (what its index is)

If we know an item is in a list we can use the .index() function to find out where in the list it is. We do need to be careful though - if we ask for the index of an item that isn't in the list, our code will break and we will get an error.

To get around this issue we can simply use some code similar to the code above to check that the item is actually in the list before we try to look for its index.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

foods = ["Pizza", "Ice Cream", "Apple", "Cherry", "Pie", "Steak"]

print(foods.index("Pie"))
