PYTHON BASICS

CONSTANTS

Continuing from variables, let's discuss constants. In mathematics, a constant is the opposite of a variable - it is a value that never changes.

Given that constants are the opposite of variables in a sense, in Python, constants are denoted with upper-case letters. All other conventions surrounding variables apply e.g. MAX_CHARS.

In general, it is best practice when programming to avoid any hard-coded values and instead use constants as much as possible. This makes the value represented by a constant easy to change if need be - all you have to do is change the value once rather than wasting time finding all instances of that value in your program.

Example Code

Copy this code to your IDE, save it and run it in the module to see what happens:

PI = 3.14
r = 20.7

print(r)

area = PI * (r*r)

print(area)
